terrorgroups.yara

username.yara
  - A rule set for looking for common username strings
  
weapon.yara
  - A rule set for trackin usage of words assocaited with weapons 

hategroups.yara
  - A rule set for tracking strings assocaited for hate groups
 
 
Hasmatlist.yara
